AgroRivero

Soluciones de apoyo para el agro

Quienes somos

AgroRivero brinda soluciones para productores de todo Uruguay mediante monitoreo de cultivos, análisis de datos productivos e interpretación de imágenes satelitales.

Con presencia principalmente en Durazno, Tacuarembó, Flores y Florida se realiza seguimiento técnico y visitas a campo para mejorar la toma de decisiones productivas.

Análisis NDVI entre años

Comparación del índice NDVI entre diferentes campañas agrícolas para evaluar el desarrollo del cultivo, identificar variaciones productivas y detectar zonas con mayor o menor vigor dentro del lote.

Sobre mí

Mi nombre es Joaquín Rivero y soy el fundador de AgroRivero. El objetivo del proyecto es acercar herramientas tecnológicas y análisis de datos al sector agropecuario para mejorar la gestión productiva.

El trabajo se enfoca en productores de todo el país, con mayor presencia en Durazno, Tacuarembó, Flores y Florida.
